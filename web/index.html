What is face morphing?

Morphing is a special effect in motion pictures and animations that changes (or morphs) one image or shape into another through a seamless transition. The face morphing algorithm morphs between faces using a common set of feature points. To morph between two faces, you need to warp both faces to a common shape so they can be blended together.

Method

You can find the details here, but the basic idea is as follows:

  1. Defining Correspondences: Find point-correspondences between the faces to be aligned using facial landmarks in the two input images.

  2. Delaunay Triangulation: You need to provide a triangulation (e.g., Delaunay triangulation) of these points that will be used for morphing.

  3. Computing the Mid-way (morphed) Face: Computing the average shape, warp both faces into that shape (calculate the affine transforms using the triangles in source image and the corresponding ones using the morphed image and warp the points inside the triangles and alpha-blend the warped images to obtain the final morphed image.
